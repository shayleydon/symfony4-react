# burstsms

### Setup

```bash
$ cd burstsms/server
$ php composer install
$ cd burstsms/ui
$ npm install
$ npm run build
```
<b>Root Directory:</b><br />
burstsms/public/

<b>Config:</b><br />
burstsms/public/.env <br />
SMS_KEY= <br />
SMS_SECRET= <br />
BITLY_ACCESS_TOKEN= <br />

<b>Server application entry point:</b><br />
burstsms/public/index.php

<b>Optional:</b><br />
Start PHP server in root directory.
burstsms/public/
